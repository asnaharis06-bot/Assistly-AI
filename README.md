# 🤖 Assistly AI — AI-Powered Chatbot
### CodeAlpha Cloud Computing Internship | Task 3

A fully functional AI-powered chatbot built with HTML, CSS, JavaScript and OpenRouter API. Supports real-time AI responses, custom training data, intent pattern matching, and detailed analytics.

---

## ✨ Features

* 🤖 **Generative AI responses** powered by OpenRouter (free tier)
* 🧠 **Custom training** with predefined Q&A pairs and intent patterns
* 📊 **Analytics dashboard** with accuracy tracking and conversation logs
* 🌐 **Website integration** with ready-to-copy embed code
* 💬 **Instant responses** with typing animation
* 🎯 **Pattern matching engine** — answers without calling AI when trained
* 📱 **Fully responsive** on mobile and desktop
* 🔧 **Customizable personality** via system prompt

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| `HTML/CSS/JavaScript` | Frontend UI |
| `OpenRouter API` | Free AI model provider |
| `Mistral / LLaMA / Auto` | Underlying AI language model |
| `localStorage` | Saving training data and settings |

## 🚀 How to Run

1. Download `Assistly AI.html`
2. Open in VS Code
3. Right-click → **Open with Live Server**
4. Get a free API key from [openrouter.ai](https://openrouter.ai)
5. Paste the key in the API Key box and click **Save**
6. Start chatting!
